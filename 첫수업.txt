Character Set
한글은 UTF-8사용할거임 무조건
UTF-8은 가변형에 유니코드 사용
UTF-16은 고정형

ASCII코드
1바이트인데 7개비트씀
127개로 엔간한거 다 표시하는데(키보드자판) 이걸로 안되는게 생겨서 만든게 유니코드

한글을 입력하면 유니코드로 바뀜 UTF-8에 따라서 ->인코딩
유니코드를 다시 한글(알아볼수있는형태)로 ->디코딩

명령창에서 /는 절대경로임 C:/myproject/main 이런거
.은 현재 나의 위치
cd ./ 하면 나의 위치부터
cd / 하면 젤 상위폴더부터
..하면 바로 위 상위폴더로 이동
cd ../../d/x.jpg  	//내 상위 두개 위 폴더로 가서 d로 가서 x.jpg찾기

파일 이름에서 *은 갯수가 0개던 몇개던 그냥 쓸수있음 (*동 *길동 *홍길동)
?는 무조건 1글자 (?길동 홍?동 홍길?)

변수명 선언방법 (3가지)
camelCase(변수명 처음(camel)은 소문자 다음단어(Case)들어갈때는 대문자로 표기하는 방식)   ->  의도적으로 카멜케이스를 사용해야하려고 노력해야함
snake_case(단어 사이의 공백을 _로 대체하는 표기법)
PascalCase
어디에 쓰이는지 정확히 알 수 있게 명확하게 짓는추세(myUserNumberCount)

많이 사용하는 자료구조 4가지
Array 				//중복허용

Map	키 : 값, '김형민' : 25, 	//처럼 기록하는거를 Map형 자료구조 중복허용
Set	배열이랑 모양은 같다	//중복불가
